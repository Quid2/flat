dBEBits8 :: Int -> Get Word8 Source #

Return the n most significant bits (up to maximum of 8)

The bits are returned right shifted: >>> unflatWith (dBEBits8 3) [0b11100001::Word8] == Right 0b00000111 True

data ConsState Source #

A special state, optimised for constructor decoding. It consists of: The bits to parse, top bit being the first to parse (could use a Word16 instead, no difference in performance) The number of decoded bits Supports up to 512 constructors (9 bits)
